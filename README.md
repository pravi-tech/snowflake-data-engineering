# Snowflake Data Engineering Project

## Overview
This project demonstrates an end-to-end Snowflake data pipeline
using AWS S3, SQL, and Python.

## Architecture
S3 → Snowflake Stage → Landing Tables → Transformed Tables

## Tech Stack
- Snowflake
- AWS S3
- Python
- SQL

## Features
- Table creation using Snowflake SQL
- Bulk data loading using COPY command
- Python-based Snowflake connectivity
